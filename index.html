Tema 2.Define los elementos que componen la estructuración de una página web:

  • Contenedor (wrapper):

    • Un contenedor (wrapper) es un concepto común en diseño y desarrollo web, utilizado para envolver y organizar elementos dentro de una página. Se refiere a un elemento HTML (generalmente un "div") que actúa como un contenedor para otros elementos, facilitando la gestión de su diseño y estilo.

  • Cabecera (Header):

    La cabecera de una página web, representada generalmente por la etiqueta "header", contiene información introductoria o elementos de navegación. Es una de las primeras secciones que los visitantes ven y suele incluir el logotipo, el título de la página y el menú principal.

  • Contenido (Content):

    El contenido de una página web, usualmente representado por la etiqueta "main", es la parte principal que muestra la información que los usuarios están buscando. Incluye texto, imágenes, videos, y cualquier otro medio relevante.

  • Menú (Menu):

    El menú es una lista de enlaces que permite a los usuarios navegar por las diferentes secciones de una página web. Puede estar incluido en la cabecera o ser un elemento independiente.

  • Pie (Footer):

    El pie de una página web, representado generalmente por la etiqueta "footer", contiene información adicional, como derechos de autor, enlaces a políticas de privacidad, y contacto. Se encuentra al final de la página.

  • Lateral (Sidebar):

    La barra lateral o sidebar es una sección que se utiliza para mostrar contenido adicional, como enlaces, anuncios, o widgets. A menudo se coloca a la izquierda o derecha del contenido principal.

  • Etiqueta Div:

    La etiqueta "div" es un contenedor genérico en HTML que se utiliza para agrupar elementos y aplicar estilos o scripts. No tiene un significado semántico por sí misma, pero es muy útil para crear estructuras complejas.
